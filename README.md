# Public Research Subset

This repository contains an anonymized research subset of the IELTS dataset.

## Included
- Blurred Part 1 videos in `subset/videos_blurred/`
- Authoritative labels from `last_label.csv`
- Emotion-related JSON files filtered to the selected subset
- Metadata files describing the subset

## Not included
- Full dataset
- Unblurred raw videos
- Personally identifying metadata

## Subset summary
- Total persons in source dataset: 218
- Selected persons in public subset: 44
- Videos processed: 44
- Label rows kept: 40

## Note
All public videos were processed with automatic face blurring. Please manually inspect them before release.
